STEP 15 — Create Docker Compose

Back to root.

Create:

docker-compose.yml

Add:

services:

  postgres:
    image: postgres:16
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: password
      POSTGRES_DB: enterprise_rag
    ports:
      - "5432:5432"

  redis:
    image: redis:7
    ports:
      - "6379:6379"
STEP 16 — Run Docker
docker compose up -d
STEP 17 — Verify Containers
docker ps

You should see:

postgres
redis
STEP 18 — Create Backend ENV

Create:

backend/.env

Add:

DATABASE_URL=postgresql://postgres:password@localhost:5432/enterprise_rag

PINECONE_API_KEY=your_key

PINECONE_INDEX=enterprise-rag

GROQ_API_KEY=your_key
